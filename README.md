# Аналіз розрахунків та висновки

За результатами проведених розрахунків інтегралу, було отримано наступні значення:
-	функцією quad з підмодуля integrate: 2.666666666666667
-	за допомогою методу Монте-Карло (за 100 експериментів): 2.6676213333333334
-	абсолютна похибка складає: 0,000954667

Як ми бачимо з порівняння отриманих результатів абсолютна похибка складає 0,000954667, тобто, на мій погляд, ми можемо зробити висновок що обчислення значення інтеграла функції за допомогою методу Монте-Карло є достатньо ефективним, враховуючи простоту реалізації даного методу.
Оптимальна точність розрахунків за методом Монте-Карло досягається на кількості 200-250 експериментів, подальше збільшення експериментів залишає точність розрахунків приблизно на одному рівні, але збільшує час розрахунку.
